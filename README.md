## Sunny's WordPress Junks Cleaner ##

- ลบ User ที่มีคำสแปมใน Display Name เช่น cash, money, bonus โดยผู้ดูแลสามารถเพิ่ม blacklist เองได้
- ล้างข้อมูลสถิติที่เป็นขยะ (ไม่มีความสัมพันธ์กับตารางอื่น)

## วิธีติดตั้งปลั้กอิน ##
ไปที่ <code>/wp-admin/plugin-install.php</code> และอัปโหลดไฟล์ zip ของปลั้กอิน จากนั้นสามารถ <b>Activate</b> แล้วเริ่มใช้งานปลั้กอินจากเมนูได้เลย !

### ระบบลบความสัมพันธ์ขยะในตาราง ###

สามารถเพิ่มความเร็วในหน้า Dashboard ได้จริง และมีนัยยะสำคัญ

## ระบบ Block API Calling ##

บล็อกการทำงานของ API ในส่วนที่ไม่จำเป็น เช่น การเช็ค Version ของ WordPress และปลั้กอินต่าง ๆ เบื้องหลัง ทุกครั้งที่เปิดหน้า Back-office (ช่วยลดเวลาโหลดได้มาก หากเซิร์ฟเวอร์มีความเร็วช้า)
